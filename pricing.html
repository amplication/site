We're in public alpha and free to use

Amplication is an open-source project built by developers for developers. Our main objective is to provide a free lifetime "community developer license" to every developer around the world.


Hosting

Amplication code is available at GitHub public repository, it can be deployed locally following project readme instructions.

Amplication hosted service is available for everyone for free as well.
Every user account has by default a free sandbox deployment for each app built with Amplication.
The app sandbox has limited resources and can reset at any time, please do not use it for anything else other than development and testing.
Remember, you can always export your code or a Docker container and deploy it elsewhere for production purposes.
In the future, we will offer paid hosting services.

Plans

Community developer plan

We believe in freedom and knowledge sharing therefore we generate actual code you can own and we built Amplication as an open-source project. Community Developer license will share developed apps by default so others can browse through all projects, copy, and customize themselves.

Pro Developer License

Developer who registers to Amplication while we are in Alpha or Beta will receive 1 year "Pro Developer license" for free starting from our official launch.
Advantages of Pro Developer license:

Option to develop both private apps and public shared apps.
Discount towards Amplication cloud hosting solution when it becomes available.
Developers collaboration.



Currently, all registered users are automatically assigned as Pro Developer licenses.
